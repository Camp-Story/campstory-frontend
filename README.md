# Run Locally

1. Node.js 설치 (호환성을 위해 v18 또는 v20 권장)
2. 프로젝트 클론

```bash
git clone https://github.com/Camp-Story/campstory-frontend.git
```

3. 프로젝트 진입

```bash
cd campstory-frontend
```

4. 패키지 설치

```bash
npm install
```

5. 서버 실행

```bash
npm run dev
```

- lint 확인

```bash
git run lint
```
